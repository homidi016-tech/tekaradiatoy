
# ویو شاپ — سایت استاتیک سبک و شیک

این پروژه یک وب‌سایت استاتیک مینیمال برای فروش خدمات تلگرام با برند **ویو شاپ** است.
بدون نیاز به بک‌اند، آماده انتشار روی GitHub Pages، Netlify یا Vercel.

## سفارشی‌سازی سریع
1) داخل `script.js` مقادیر زیر را با اطلاعات خودتان عوض کنید:
```js
const TELEGRAM_USERNAME = "your_username"; // بدون @
const WHATSAPP_NUMBER = "0098XXXXXXXXXX"; // با کد کشور
const PHONE_NUMBER = "+98XXXXXXXXXX";
```
2) محصولات را در `products.json` ویرایش کنید (عنوان، تصویر، قیمت و گزینه‌ها).
3) لوگو را با فایل `assets/logo.svg` جایگزین کنید (اختیاری).

## انتشار
- **GitHub Pages**: ریپو بسازید، فایل‌ها را پوش کنید، از تنظیمات ریپو Pages را روی branch `main` و پوشه `/` فعال کنید.
- **Netlify**: پوشه را Drag & Drop کنید.
- **Vercel**: پروژه جدید بسازید و همین پوشه را آپلود کنید.

## نکات
- همه چیز RTL و واکنش‌گراست.
- تم تاریک/روشن دارد.
- خرید از طریق دکمه‌های تلگرام/واتس‌اپ به صورت چت مستقیم است.
